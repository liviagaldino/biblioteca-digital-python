# Sistema de Gerenciamento de Biblioteca Digital

## Descrição

Sistema desenvolvido em Python para auxiliar uma biblioteca universitária no gerenciamento de documentos digitais.

## Funcionalidades

- Listar documentos
- Adicionar documentos
- Renomear documentos
- Remover documentos
- Organizar documentos por tipo de arquivo
- Organizar documentos por ano de publicação

## Estrutura do Projeto

biblioteca-digital-python/

├── documentos/

├── testes/
├── README.md
├── CONTRIBUTING.md
├── relatorio_testes.md

- Git
- GitHub

## Como Executar

1. Acesse a pasta src

2. Execute:

python main.py
